# Один EXE для Windows — скинуть другу

Цель: **один файл** установщика (.exe), который можно отправить другу; он открывает его на любом Windows ПК, устанавливает приложение, запускает — и в один клик разворачивает свой узел и кошелёк.

## Как получить этот один EXE

### Вариант 1: GitHub Actions (без своего Windows)

1. Залейте проект в GitHub (если ещё не залит).
2. Откройте вкладку **Actions** → workflow **Build Windows EXE**.
3. Запустите его вручную (**Run workflow**) или дождитесь автоматического запуска при пуше в `main`/`master`.
4. По окончании скачайте артефакт **windows-exe** (ZIP).
5. Распакуйте ZIP — внутри будет один файл вида **ELENA Wallet_0.1.0_x64-setup.exe** (или с другой версией). Это и есть тот самый один EXE.

Этим файлом можно делиться: отправить другу, выложить в общий доступ. Друг скачивает, запускает на своём Windows ПК — устанавливается приложение.

### Вариант 2: Сборка на своём Windows ПК

1. Клонируйте/скопируйте репозиторий на компьютер с Windows.
2. Установите [Node.js](https://nodejs.org/), [Rust](https://rustup.rs/) и [Visual Studio Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/) (компоненты для C++).
3. В корне репозитория откройте командную строку в папке `elena-desktop` и запустите:
   ```
   build-windows.bat
   ```
   (скрипт лежит в `elena-desktop/build-windows.bat`; его можно запускать из корня репозитория, указав путь к нему.)
4. После сборки один установочный EXE будет в:
   `elena-desktop\src-tauri\target\release\bundle\nsis\ELENA Wallet_0.1.0_x64-setup.exe` (имя может отличаться по версии).

Этот один EXE можно скидывать друзьям.

## Что делает друг, получивший EXE

1. Скачивает файл (например, `ELENA Wallet_0.1.0_x64-setup.exe`).
2. Запускает его на любом Windows ПК (может потребоваться «Запуск от имени администратора» или подтверждение UAC).
3. Проходит шаги установки (папка, ярлык и т.д.) и завершает установку.
4. Запускает **ELENA Wallet** из меню Пуск или с рабочего стола.
5. В приложении нажимает **«Запустить узел»** — создаётся кошелёк (если первый раз), запускаются узел и gateway.
6. Нажимает **«Открыть кошелёк»** — в браузере открывается веб-кошелёк по адресу http://localhost:9180.

После этого у него свой узел и свой кошелёк, без отдельной установки программ.

## Требования на ПК друга

- Windows 10 или 11 (64-bit).
- Обычно уже есть WebView2 (часто ставится с обновлениями Windows). Если приложение попросит — установить WebView2 по подсказке.
- Доступ в интернет не обязателен для работы узла после установки; нужен только для первого запуска узла в сети (если узел подключается к пирам).
